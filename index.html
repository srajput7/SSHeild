SSHield

A Simple Tool to Audit and Enhance SSH Security

About SSHield

SSHield is a lightweight bash script designed to help system administrators audit and improve their SSH configurations. The tool checks for common vulnerabilities and suggests fixes to strengthen your server's security.

How to Use

  1. Download the script from the GitHub repository.
  2. Make it executable using the command: chmod +x SSHield.sh
  3. Run the script with: ./SSHield.sh

Features

  • Checks for root login permissions.
  • Verifies if password authentication is enabled.
  • Ensures SSH Protocol 2 is used.

Script Contents

        #!/bin/bash
        # SecureShellCheck: SSH Security Audit Tool
        echo "SecureShellCheck: Auditing SSH Configuration..."
        CONFIG_FILE="/etc/ssh/sshd_config"
        if grep -q "^PermitRootLogin yes" $CONFIG_FILE; then
            echo "[WARNING] Root login is enabled. Disable it for better security."
        else
            echo "[OK] Root login is disabled."
        fi
        if grep -q "^PasswordAuthentication yes" $CONFIG_FILE; then
            echo "[WARNING] Password authentication is enabled. Consider using key-based authentication."
        else
            echo "[OK] Password authentication is disabled."
        fi
        if grep -q "^Protocol 1" $CONFIG_FILE; then
            echo "[WARNING] SSH Protocol 1 is outdated and insecure. Use Protocol 2."
        else
            echo "[OK] Using SSH Protocol 2."
        fi
        echo "Audit complete. Review the above findings."
