SSHield

A Simple Tool to Audit and Enhance SSH Security

About SSHield

SSHield is a lightweight bash script designed to help system administrators audit and improve their SSH configurations. The tool checks for common vulnerabilities and suggests fixes to strengthen your server's security.

How to Use

  1. Download the script from the GitHub repository.
  2. Make it executable using the command: chmod +x SSHeild.sh
  3. Run the script with: ./SSHeild.sh

Features

  • Checks for root login permissions.
  • Verifies if password authentication is enabled.
  • Ensures SSH Protocol 2 is used.

Contribute

If you have ideas to improve SSHield, feel free to contribute by creating an issue or submitting a pull request on our GitHub repository.
